XDA001 Web BLE Console

Commands List

Command Response Description
AT OK check the connection
ATR reset then and disconnect reset the device
ATH command list help to show the supported commands
ATI device information show the device information
ATLF get file List List all files in LittleFS
ATRF[=filename] get file content get file content, example:ATRF=/test.txt
ATDF[=filename] delete file delete file, example:ATDF=/test.txt
AT+TEMP[=params] get and set refer temperature to offset the value get temperature, example:AT+TEMP=? , set temperature for calculating the offset, example:AT+TEMP=25.7, xda001 use 25.7 (degC) as the reference to calculate the offset
AT+INTV[=params] get and set interval upload time gap get interval time, example:AT+INTV=? , set interval, example:AT+INTV=60 , interval time is 60 seconds
